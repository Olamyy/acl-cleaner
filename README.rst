acl-cleaner
===========

Python tool for converting bibtex to the canonical ACL anthology format.

Usage
-----

Installation
------------
Clone the repository and run ``pip install -e .``

You can then run the tool by calling ``acleanbib [BIBTEX] [--OUTPUT]``

Requirements
^^^^^^^^^^^^
1. click
2. bibtexparser
3. pandas
